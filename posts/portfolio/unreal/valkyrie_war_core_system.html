Valkyrie War 코어 시스템 설계

안정적인 게임 흐름과 체계적인 데이터 관리를 위해 언리얼 엔진의 GameInstance 및 Subsystem을 적극 활용하여 구현한 매니저 시스템 구조입니다.

GameManager

Inherits: UGameInstance
  • 전역에서 공통으로 사용되는 테이블 데이터 및 위젯 등의 데이터 보관 및 접근 포인트 제공.
  • 아이템, 캐릭터 등 게임 내 주요 객체들의 고유 식별자(UID) 생성 역할 수행.

DataManager

Inherits: UGameInstanceSubsystem
  • 정적인 테이블 데이터와 동적인 저장 데이터를 메모리에 적재하여 런타임에 활용.
  • FTableRowBase를 활용한 테이블 데이터 사용
  • 각 데이터를 Module이라는 단위로 보관하여 효율적으로 관리.
  • 예시) UItemModule : UObject 형태를 통해 기능별 데이터 응집도 향상.

LevelManager

Inherits: UGameInstanceSubsystem
  • 끊김 없는 사용자 경험을 위한 비동기 레벨 로드(Async Level Loading) 기능 구현.
  • 다양한 레벨(로그인, 로비, 전투 등) 간의 전환 및 트랜지션 처리 전담.

SaveManager

Inherits: UGameInstanceSubsystem
  • 언리얼 엔진의 기본 클래스인 USaveGame을 기반으로 세이브 데이터를 관리.
  • 로컬 저장 데이터의 보안을 위해 데이터를 직렬화(Serialization)하고 AES 암호화를 적용하여 저장 및 로드 수행.

UIManager

Inherits: UGameInstanceSubsystem
  • UI 위젯을 동적으로 생성하고, 불필요한 파괴를 막기 위해 메모리에 캐싱(Caching)하여 관리.
  • 통일된 템플릿(인터페이스) 구조를 사용하여 다양한 형태의 위젯을 일관된 로직으로 여닫아 주는 기능 구현.

WorldEventSystem

Inherits: UWorldSubsystem
  • 월드 내에서 발생하는 각종 Delegate(이벤트 콜백)를 중앙에서 통합하여 관리.
  • 다수의 팀원이 작업할 때 발생할 수 있는 이벤트 중복 선언 및 호출을 방지하기 위한 목적.
  • 충돌 방지 구조: 동시 수정 시 발생하는 Git 충돌(Conflict)을 최소화하기 위해, 시스템 내부에 구조체(Struct) 단위로 이벤트 정의를 모듈화하여 분리.
